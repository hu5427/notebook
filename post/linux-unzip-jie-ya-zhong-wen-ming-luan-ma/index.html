找到解决方案
# -O字符集为DOS, Windows和OS/2档案指定字符编码
 -O CHARSET  specify a character encoding for DOS, Windows and OS/2 archives
使用以下命令
unzip -O CP936 test.zip

或者

unzip -O gbk test.zip
